# FishSpot

JavaScript（Node.js / Express）で作成したWebアプリです。  
おすすめの釣りスポットを登録・共有できるようにイメージしたポートフォリオ用アプリです。

## デモ
https://blooming-island-48446-7dae472217b3.herokuapp.com/

![FishSpot](docs/screenshot.png)

## 使用技術
- JavaScript
- Node.js
- Express
- EJS
- Heroku

## 機能
- トップページ表示
- 一覧表示
- 詳細ページ表示

## 環境変数
本アプリケーションでは `.env` ファイルを使用しています。  
機密情報は GitHub には含めていません。

機能追加・画面改善を行ったものです。
